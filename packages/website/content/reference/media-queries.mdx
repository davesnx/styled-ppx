# Media queries and other at-rules

Declaration blocks (`[%css]{:reason}`, `[%styled.<tag>]{:reason}`) support nested at-rules, especially:

- `@media`
- `@supports`
- `@container`

```css
display: grid;

@media (min-width: 768px) {
  grid-template-columns: repeat(2, 1fr);
}

@supports (display: grid) {
  gap: 16px;
}

@container card (width >= 400px) {
  padding: 24px;
}
```

At-rules nest under selectors (and vice versa) and render correctly scoped: `&:hover` inside `@media` becomes `@media (...) { .generated:hover { ... } }{:styled-ppx-css}`.

In [`[%styled.global]`](./globals), top-level at-rules work exactly as in a plain stylesheet (`@media (prefers-reduced-motion: reduce) { * { animation: none; } }{:styled-ppx-css}`).

If you need keyframes, use the dedicated [keyframe extension](./keyframe) instead of writing them inline in a declaration block.

## Limitation: no interpolation in preludes

The at-rule **prelude** (the condition part) must be literal CSS. `@media (max-width: $(breakpoint)){:styled-ppx-css}` is a compile error:

```
Interpolation in @media preludes is not supported during static extraction.
CSS custom properties (var()) are not valid in media query conditions.
Inline the value directly.
```

Interpolated values become CSS custom properties in the extracted stylesheet, and per the CSS spec `var()` cannot appear inside media-query conditions. So there is no mechanism to supply the value at runtime. This applies to `@media`, `@supports`, and `@container` preludes alike. (Interpolation inside the at-rule's *body* is fine: `@media (min-width: 768px) { gap: $(spacing); }{:styled-ppx-css}`.)

What works instead:

- **Write the condition literally.** Breakpoints are design decisions that rarely need to be runtime values.
- **If a breakpoint truly must vary**, define one block per variant and switch in code:

```reason
let compact = [%css {|
  @media (max-width: 480px) { display: none; }
|}];
let comfortable = [%css {|
  @media (max-width: 768px) { display: none; }
|}];

let styles = density == `Compact ? compact : comfortable;
```

```mlx
let compact = [%css {|
  @media (max-width: 480px) { display: none; }
|}]
let comfortable = [%css {|
  @media (max-width: 768px) { display: none; }
|}]

let styles = if density = `Compact then compact else comfortable
```
