# Melange

Ensure that you have set up a Melange project beforehand. If you haven't done so yet, refer to the Melange getting started [documentation](https://melange.re/v3.0.0/getting-started.html).

This guide assumes that you'll use opam and dune.

### Install

```bash
opam install styled-ppx
```

### Setup

Add `styled-ppx` under dune's preprocess pps and `styled-ppx.melange` under libraries, both in your library with `(modes melange)` and in the `melange.emit` stanza:

```diff
(library
  (name ...)
  (modes melange)
  (libraries
+   styled-ppx.melange
    reason-react)
  (preprocess
    (pps
+     styled-ppx
      melange.ppx
      reason-react-ppx)))
```

Then add one rule that collects the extracted CSS into a stylesheet, and link it from your HTML:

```dune
(rule
 (mode promote)
 (target styles.css)
 (deps
  (glob_files_rec *.pp.ml))
 (action
  (run styled-ppx.generate --output %{target} %{deps})))
```

<Callout type="warning">
  Without this rule your app compiles fine but renders unstyled: all CSS
  lives in the generated stylesheet. See
  [Setting up the dune rules](/getting-started/dune-setup) for the full
  reference: rule placement, the `melange.emit` stanza, the CLI, sharing code
  with a native server, and production configuration.
</Callout>

### Example

```reason
/* This is a ReasonReact component with those styles applied */
module Link = [%styled.a (~color=CSS.hex("4299E1")) => {|
  font-size: 1.875rem;
  line-height: 1.5;
  text-decoration: none;
  margin: 0px;
  padding: 10px 0px;
  color: $(color);
|}];

/* This is a CSS.styles value carrying the generated classNames */
let layout = [%css {|
  display: flex;
  width: 100%;
  height: 100%;
  justify-content: center;
  align-items: center
|}];

/* Later in a component: pass it via the `styles` attribute */
<div styles=layout>
  <Link
    color={CSS.hex("333333")}
    href="https://sancho.dev"
    rel="noopener noreferrer">
    {React.string("sancho.dev")}
  </Link>
</div>;
```

Learn more about [`[%css]`](/reference/css), the [`styles` attribute](/reference/css#the-styles-attribute), and [styled components](/reference/styled-components).
