# ReScript

<Callout type="error">
  **ReScript is no longer supported.** The ReScript runtime was removed in
  styled-ppx 0.62.0, together with the `@davesnx/styled-ppx` npm package
  workflow.
</Callout>

styled-ppx focuses on **OCaml and Reason**, targeting [Melange](https://melange.re) for the browser and [server-reason-react](https://github.com/ml-in-barcelona/server-reason-react) for native server rendering. Maintaining a third compiler target with a diverging language and build system wasn't sustainable, and the static-extraction architecture introduced in 0.62.0 relies on dune.

## What are my options?

- **Migrate to Melange.** Melange compiles OCaml/Reason to JavaScript with first-class React support and is styled-ppx's primary target. See [Getting started with Melange](/getting-started/melange).
- **Stay on ReScript.** The last release with ReScript support was **0.61.0**, installable from npm as `@davesnx/styled-ppx`. It keeps working, but won't receive updates and predates static extraction.

If you have questions, reach out on [GitHub issues](https://github.com/davesnx/styled-ppx/issues).
