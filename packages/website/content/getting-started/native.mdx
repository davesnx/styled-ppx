# OCaml (native)

This guide assumes that you'll use [opam](https://github.com/ocaml/opam) and [dune](https://github.com/ocaml/dune) and have a server or native process running, rendering HTML with [server-reason-react](https://github.com/ml-in-barcelona/server-reason-react).

### Install

```bash
opam install styled-ppx
```

### Setup

Add `styled-ppx` with the `--native` flag under dune's preprocess pps (it switches the generated component code to target server-reason-react), and `styled-ppx.native` as a library:

```diff
(library
  (name ...)
  (libraries
+  styled-ppx.native
   server-reason-react.react
   server-reason-react.reactDom)
  (preprocess
   (pps
+   styled-ppx
+   --native
    server-reason-react.ppx
    server-reason-react.melange_ppx)))
```

Then add one rule that collects the extracted CSS into a stylesheet:

```dune
(rule
 (mode promote)
 (target styles.css)
 (deps
  (glob_files_rec *.pp.ml))
 (action
  (run styled-ppx.generate --output %{target} %{deps})))
```

<Callout type="warning">
  Without this rule your pages render unstyled: all CSS lives in the
  generated stylesheet. See
  [Setting up the dune rules](/getting-started/dune-setup) for the full
  reference: rule placement, the CLI, sharing UI with a Melange client, and
  production configuration.
</Callout>

### Example

Serve `styles.css` like any static asset and link it in the document you render:

```reason
module Document = {
  [@react.component]
  let make = (~children) => {
    <html>
      <head>
        <link rel="stylesheet" href="/styles.css" />
      </head>
      <body>
        <main styles=[%css "max-width: 60ch; margin: 0 auto"]>
          children
        </main>
      </body>
    </html>;
  };
};

/* Let's assume we are using dream (from https://github.com/aantron/dream) */
let some_server_side_handler = _request => {
  Dream.html(ReactDOM.renderToString(<Document> <App /> </Document>));
};
```



Learn more about [`[%css]`](/reference/css), [global styles](/reference/globals) (`@font-face`, resets, `:root` variables), and [sharing UI with a Melange client](/getting-started/dune-setup#sharing-ui-between-client-and-server).
