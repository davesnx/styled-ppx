# Releases

`styled-ppx` versioning works like semver but without major releases. The
major version is always 0, like `0.x.x`. Each minor version can contain
breaking changes.

The full release history, with notes for every version, lives on GitHub:

**[github.com/davesnx/styled-ppx/releases](https://github.com/davesnx/styled-ppx/releases)**

The changelog is also available in the repository as
[`CHANGES.md`](https://github.com/davesnx/styled-ppx/blob/main/CHANGES.md).
For the changes in 0.62, see the [migration guide](/migrate-to-v062).
